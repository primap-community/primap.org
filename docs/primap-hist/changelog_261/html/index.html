Overview

This analysis shows per country changes from PRIMAP-hist PRIMAP-hist_v2.6_final_nr to PRIMAP-hist PRIMAP-hist_v2.6.1_final_nr on a per country, gas, and sector level. It aims to give quick overview over the countries with the highest changes and presents details on thesectors and gases with the highest changes. Where changes are high and / or impact cumulative emissions we give explanations for the change.

In the following we give a short overview over the most importantchanges in input data and processing which will help to understand per country, gas, and sector changes.For general information on the dataset please consult the data description document.

Overview over changes from v2.6_final to v2.6.1_final

The v2.6.1 release updates country reported and some third party data. New and updated input data sources are UNFCCC CRT data from BTRs for AnnexI and non-AnnexI countries, FAOSTAT agricultural and LULUCF data, EDGAR data for all gases, and cement process emissions data. For detailed per country changes we refer to the detailed changelog available on the primap website.

Changes in data sources and preprocessing (v2.6.1)

Data source updates (v2.6.1)

  • CRF / BTR / AnnexI inventories: We have read country reported AnnexI data from Biannial Transparency Reports (BTR):
    • For almost all AnnexI countries we have now included the CRT data from BTR1 which covers 1990-2022 and in some cases up to 3 years before 1990. We have made the following exceptions:
      • Belarus: CRT data does not contain f-gas emissions, thus we continue to use the inventory data used in v2.6 for f-gases.
      • Poland: Data for 1989 is missing in the CRT files (1988 is present). We use 1988 data from the EU data used in v2.6.
      • Sweden: In the CRT data several sectors are marked as confidential and their emissions are missing in the sum for some years. This makes the data unusable and we continue to use the EU data.
    • We have included CRT data for the following non-AnnexI countries: Algeria, Argentina, Azerbaijan, Brazil, Brunei-Darussalam, Bhutan, Chile, China (Mainland, Hong Kong, Macau), Côte d’Ivoire, Colombia, Ecuador, Egypt, Georgia, Ghana, Guinea Bissau, Guyana, Indonesia, Kenya, Lebanon, Morocco, Maldives, Mauritius, Malaysia, Namibia, Nigeria, Panama, Paraguay, Singapore, Serbia, South Africa, Tajikistan, Tunisia, Uruguay, Uzbekistan, Vatican City State, Venezuela, Zimbabwe). For details on data coverage and use, please consult the detailed changelog.
  • UNFCCCDI: There is no new information since the last update
  • UNFCCC BUR, BTR, and NC data (non-AnnexI) We have not included any new submissions.
  • Andrew cement emissions data have been updated to the February 2025 version.
  • FAOSTAT data have been updated to the November 2024 version. We also updated data reading which is now PRIMAP2 based: github.

Changes in PRIMAP-hist source creation (v2.6.1)

Methodology changes (v2.6.1)

  • We have changed the extrapolation method for CH4 and N2O data in M.LULUCF from linear to constant to avoid high high growth rates when extrapolation from only a few data points.

Data source prioritization (v2.6.1)

  • Replacement with updated version of the data. See Section Changes in data sources and preprocessing (v2.6.1) above.

Sectors and gases (v2.6.1)

  • no changes

Composite source generation methodology (v2.6.1)

  • No changes

Special treatment of individual countries (v2.6.1)

  • Special treatments for Albania (1.B.1, 1.B.2, 2.C), Croatia (1.B.2), Dominican Republic (2.C), The Democratic Republic of the Congo (1.A), France (1.B.2), Germany (1.B.2), Ghana (1.B.2), Jamaica (1.B.1), Jordan (1.B.2), Latvia (2.C), Myanmar (2.B), Namibia (2.A), Nigeria (2.C), Pakistan (1.B.2), Saint Lucia (1.B.1), Saint Vincent and the Grenadines (1.A), Saudi Arabia (2.C), Vanuatu (1.B.2), and the former Soviet Union states (2.C) are still in place (see “Special treatment of individual countries (v2.6)” in the changelog pdf.
  • Special treatments for China (2.B) and Ireland (2.B) have been removed.
  • Jamaica: We extrapolate UNFCCC data in 1.B.2, CO2 for a few years to the past to better match EDGAR. We also extrapolate to the future to avoid scaling of EDGAR data.
  • Austria: We extrapolate UNFCCC data in 1.B.2, CO2 for a few years to the past to better match EDGAR.
  • Brunei Darussalam: We extrapolate UNFCCC data in 1.B.2, CO2 for a few years to the past to better match EDGAR.
  • Belgium: We remove EDGAR data in 1.B.2, CO2 before 1990 as it is incompatible with country reported data.
  • Greece: We remove EDGAR data in 1.B.2, CO2 before 1990 as it is incompatible with country reported data.
  • Tunisia: We remove EDGAR data in 1.B.2, CO2 before 1990 as it is incompatible with country reported data.
  • Netherlands: We remove EDGAR data in 1.B.2, CO2 before 1975 as it is incompatible with country reported data.
  • USA: We remove EDGAR data in 1.B.2, CO2 before 1975 as it is incompatible with country reported data.
  • Bulgaria: We extrapolate UNFCCC data in 1.B.1, CO2 for a year to the past to better match EDGAR data.
  • Moldova: We linearly extrapolate EDGAR CH4 data in 1.B.1 to avoid scaling to very high CEDS data for the long term extrapolation.
  • Singapore: We extrapolate EDGAR data for 2.B, N2O to 2023 to avoid scaling of country reported data to EDGAR.
  • Czechia: We extrapolate UNFCCC data in 1.B.1, CH4 for a years to the past to better match EDGAR.
  • Bhutan: FAO data is incomplete from 1991 to 2005, so we remove it.
  • North Macedonia: CDIAC data for 2.A is zero in 2020. But cement production has not stopped in 2020, thus we remove the zero data point.

Bug fixes and resolved issues (v2.6.1)

  • The issue with missing synthetic fertilizer emissions in the FAO dataset has been resolved. (issue #22)
  • Bugs affecting individual countries are listed on the country pages.

Known problems (v2.6.1)

This listing only contains issues that affect several countries. For individual country issues see also the detailed changelog on the primap website.

  • We have further worked on problems with scaling of historical CO2 emissions especially for 1.B.2. For some countries problems remain. (issue #34)
  • Process emissions from lime production (2.A.2) are currently not included in the third party priority (TP) scenario, as we use Andrew cement data as a proxy for sector 2.A. To include lime process emissions sub-sector resolution of 2.A would have to be introduced. (issue #38)
  • EDGAR data for N2O, 1.B.1 is very limited for some countries, thus extrapolation generates most of the data for those countries. (issue #42)
  • For some countries some timeseries start later than 1750, so long range historical data (CEDS etc.) are (partly) missing. This issues has been fixed for most cases, but a few remain. (see issue #56)
  • For problems that were found after the release of the dataset, please consult the issue tracker github.com/JGuetschow/PRIMAP-hist. If you encounter a problem please add an issue or contact the authors directly.

Noteworthy changes (v2.6.1)

Here we only list changes affecting all countries, country groups or several countries. For detailed per country changes please consult the detailed changelog available on zenodo and the primap.org website.

  • The new EDGAR 2024 data has changed emissions for many sectors. The energy sector (fugitive emissions and non-CO2 emissions in 1.A) and the waste sector are especially affected with changes that are often much higher than +100% or -50%. This leads to emissions changes for many countries, especially non-AnnexI countries, in the TP scenario. The CR scenario is less affected because EDGAR is only used to fill and extrapolate country reported data.
  • For the “country reported priority” time series we have new data covering all sectors for several countries. This leads to changed data in many sectors for these countries.
  • We replaced the EU data and other national inventory data used in v2.6 for most AnnexI countries by data reported to the UNFCCC. The data used in v2.6 might have had an impact on emissions, because emissions from these territories might be excluded for the EU reporting but included for the UNFCCC reporting (especially France, (Netherlands territories should not be included in UNFCCC emissions reporting either)).

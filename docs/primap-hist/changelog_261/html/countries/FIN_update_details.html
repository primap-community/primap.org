In the third party priority scenario we have the following changes for aggregate Kyoto GHG and national total emissions excluding LULUCF (M.0.EL):

Notes on data changes

Here we list notes explaining important emissions changes for the country.

  • CRT data has been included but replaces official data reported to the EU which does not show noteworthy differences. Changes in the CR time-series are small and come from updated third party data (EDGAR, FAO) which is used to extrapolate the country reported data.
  • Changes in the TP time-series are dominated by a 65% reduction in Waste CH4 in new EDGAR data. Smaller influences in other sectors also come from new EDGAR and FAO data. FAO data changes are limited to the latest years.

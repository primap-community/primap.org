Overview over changes

In the country reported priority scenario we have the following changes for aggregate Kyoto GHG and national total emissions excluding LULUCF (M.0.EL):

  • Emissions in 2023 have changed by -1.7%% (-7.05 Gg CO2 / year)
  • Emissions in 1990-2023 have changed by 0.7%% (2.62 Gg CO2 / year)

In the third party priority scenario we have the following changes for aggregate Kyoto GHG and national total emissions excluding LULUCF (M.0.EL):

  • Emissions in 2023 have changed by -4.9%% (-14.70 Gg CO2 / year)
  • Emissions in 1990-2023 have changed by -0.4%% (-1.25 Gg CO2 / year)

Notes on data changes

Here we list notes explaining important emissions changes for the country.

  • No new country reported data has been included. Country reported data covers only 1994, 2000, and 2005, thus the third party data has a high influence on the CR time-series.
  • The main 2023 change for both scenarios is the removal of the CH4 data for M.AG.ELV which is no longer contained in the FAO data.
  • Changes in 4 (CH4), 1.B.2 (CH4, CO2), and 5 (N2O) come directly from EDGAR in both scenarios as there is no country reported data available.

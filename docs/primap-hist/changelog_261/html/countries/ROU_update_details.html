Notes on data changes

Here we list notes explaining important emissions changes for the country.

  • CRT data has been included but replaces official data reported to the EU which does not show noteworthy differences. Changes in the CR time-series are small and come from updated third party data (EDGAR, FAO) which is used to extrapolate the country reported data.
  • Changes in the TP time-series come from updated EDGAR data. EDGAR has updated the last year data for some sectors (leading to 2023 emissions changes) and full time-series for others (leading to high cumulative emissions changes).

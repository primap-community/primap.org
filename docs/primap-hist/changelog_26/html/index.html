Overview

This analysis shows per country changes from PRIMAP-hist PRIMAP-hist_v2.5.1_final_nr to PRIMAP-hist PRIMAP-hist_v2.6_final_nr on a per country, gas, and sector level. It aims to give quick overview over the countries with the highest changes and presents details on thesectors and gases with the highest changes. Where changes are high and / or impact cumulative emissions we give explanations for the change.

In the following we give a short overview over the most importantchanges in input data and processing which will help to understand per country, gas, and sector changes.For general information on the dataset please consult the data description document.

Overview over changes from v2.5.1_final to v2.6_final

The v2.6 release updates country reported and some third party data. New and updated input data sources are UNFCCC data for AnnexI and non-AnnexI countries, official country inventories, Energy Institute’s fossil fuel related emissions, and cement process emissions data. For detailed per country changes see detailed changelog.

Changes in data sources and preprocessing (v2.6)

Data source updates (v2.6)

  • CRF / BTR / AnnexI inventories: We have read country reported AnnexI data from Biannial Transparency Reports (BTR) and several other sources:
    • For Australia we have read the official BTR1 submission
    • Canada has submitted CRF tables as additional information to the NID to BTR1
    • New Zealand has published CRF tables as part of an official inventory (but not submitted them to the UNFCCC)
    • All countries participating in the EU ETS submit GHG inventories to the EU which we have included (EEA)
    • For the USA, the United Kingdom and Japan we use national inventories covering all sectors and gases.
    • For Türkiye and Belarus we use national emissions inventories covering only the main sectors and gases and use CRF2023 to downscale the data to PRIMAP-hist categories and gases.
  • UNFCCCDI: We have updated UNFCCCDI data but there is no new information since the last update.
  • UNFCCC BUR, BTR, and NC data (non-AnnexI) We have included the following Biannial Update Report (BTR), Biannial Transparency Report (BTR) and National Communication (NC) submissions for non-AnnexI countries:
    • Algeria BUR1 / NIR: detailed data for 1990-2020
    • Argentina BUR5: detailed data for 1990 - 2020 (read from the official emissions inventory underlying BUR5)
    • Bangladesh BUR1: data for 2013 - 2019
    • Burundi BUR1: detailed data for 2005 - 2019
    • Cabo Verde, BUR1: detailed inventory for 2019, trends for 1995, 2000, 2005, 2010, 2015 which needed some downscaling
    • China, Macao, Hong Kong BUR3, NC4: Inventories for 2005, 2017, and 2018. 2005 data is downscaled to the PRIMAP-hist gases and categories.
    • Guinea BUR1: detailed data for 1990, 1995, 2000, 2005, 2010, 2015, 2018, 2019
    • India NC3: included trends (2011 - 2018) which were downscaled to PRIMAP-hist gases. 2019 data already included in v2.5.1
    • Mongolia BUR2: Data for 1990-2020 but not usable for all sectors for all years and downscaling not possible for all sectors.
    • Republic of Korea 2023 inventory: detailed data for 1990 - 2021
    • Saint Kitts and Nevis BUR1: Data for 2008-2018, downscaling needed.
    • Taiwan 2023 inventory: detailed data for 1990 - 2021
    • South Africa Inventory underlying BUR5: detailed data for 2012-2020
    • United Arab Emirates BUR1: Detailed inventory for 2021.
    • Uzbekistan BTR1 / NIR: Detailed data for 1990-2021
  • Andrew cement emissions data have been updated to the May 2024 version.
  • EI: Energy Institute’s Statistical Review of World Energy data have been updated to the 2024 version

Changes in PRIMAP-hist source creation (v2.6)

Methodology changes (v2.6)

  • To better incorporate shorter term trends into our modeling we have reduced the period that is used to calculate trend for numerical extrapolation to 5 years for all sectors and gases except energy CO2. For energy CO2 we keep the 15 year period to filter out limited time effects of the Covid-19 pandemic. (closes issue #80)

Data source prioritization (v2.6)

  • Replacement with updated version of the data. See Section Changes in data sources and preprocessing (v2.6) above.
  • For categories 4 and 5 FAOSTAT data was accidentally prioritized over EDGAR. This has been fixed. For details see section Bug fixes and resolved issues (v2.6) below.

Sectors and gases (v2.6)

  • no changes

Composite source generation methodology (v2.6)

  • No changes

Special treatment of individual countries (v2.6)

  • Special treatments for Albania (1.B.1, 1.B.2, 2.C), China (2.B), Croatia (1.B.2), Dominican Republic (2.C), The Democratic Republic of the Congo (1.A), France (1.B.2), Ghana (1.B.2), Jamaica (1.B.1), Latvia (2.C), Myanmar (2.B), Namibia (2.A), Nigeria (2.C), Pakistan (1.B.2), Saint Lucia (1.B.1), Saint Vincent and the Grenadines (1.A), Saudi Arabia (2.C), Vanuatu (1.B.2), and the former Soviet Union states (2.C) are still in place (see [Special treatment of individual countries (v2.5.1)]).
  • Ireland: We replace missing data in the EEA source for 2.B, CO2 with 0 (from CRF2023 data).
  • Germany: We extrapolate EI2024 data for CO2 in 1.B.2 into the past until 2006 to better match EDGAR data and remove a scaling problem (TP scenario only).
  • Jordan: We have updated the special treatment for CO2 in 1.B.2 to start in 1988 instead of 2014 to remove a scaling problem for pre-1990 data.

Bug fixes and resolved issues (v2.6)

  • In the UNFCCCDI data the IPCCC1996 sector 2 (Solvent and other product use) was not correctly mapped to IPCC2006 sector 2.D. This has been corrected. Where it influences country’s emissions (very few cases) it is mentioned in the detailed changelog.
  • In the UNFCCCDI data liming emissions were not correctly mapped and missing in the data used in PRIMAP-hist. The effect is small and does not significantly influence the total emissions of any country.
  • In PRIMAP-hist v2.5.1 FAOSTAT data was accidentally included for sectors 4 and 5 with higher priority than EDGAR 8.0. As FAOSTAT is based on PRIMAP-hist TP for non-AFOLU sectors this means that for the TP time-series the last PRIMAP-hist version was used instead of the new EDGAR data. We have fixed this in v2.6. As there are high changes in EDGAR 8.0 vs. older data this has influenced many country’s emissions. The influence on total emissions is below 0.5% for almost all countries. The country pages in the detailed changelog list the influence where relevant.
  • We have further worked on problems with scaling of historical CO2 emissions especially for 1.B.2. For some countries problems remain. (issue #34)
  • Bugs affecting individual countries are listed on the country pages in the detailed changelog.

Known problems (v2.6)

This listing only contains issues that affect several countries. For individual country issues see also the listing of detailed per country changes the file PRIMAP-hist_v2.5_detailed_CHANGELOG.pdf.

  • FAO data for synthetic fertilizers do not cover the full period (1961 - 2018) for several countries. In the 2020 release data for some countries were removed leading to changed emission estimates. Some time series start later than 1961, while others have gaps. We currently use the summed “Agricultural Soils” data from the FAO emissions total domain. We plan to use detailed FAO data processing in v2.5 again and solve this issue. (issue #22)
  • We have further worked on problems with scaling of historical CO2 emissions especially for 1.B.2. For some countries problems remain. (issue #34)
  • Process emissions from lime production (2.A.2) are currently not included in the third party priority (TP) scenario, as we use Andrew cement data as a proxy for sector 2.A. To include lime process emissions sub-sector resolution of 2.A would have to be introduced. (issue #38)
  • EDGAR data for N2O, 1.B.1 is very limited for some countries, thus extrapolation generates most of the data for those countries. The main problem is Croatia, however it’s not relevant on the level of aggregate emissions. (issue #42)
  • For some countries some timeseries start later than 1750, so long range historical data (CEDS etc.) are (partly) missing. This issues has been fixed for most cases, but a few remain. (see issue #56)
  • For problems that were found after the release of the dataset, please consult the issue tracker github.com/JGuetschow/PRIMAP-hist. If you encounter a problem please add an issue or contact the authors directly.

Noteworthy changes (v2.6)

Here we only list changes affecting all countries, country groups or several countries. For detailed per country changes please consult the detailed changelog available on zenodo and the primap.org website.

  • For the “third party priority” scenario (TP), input data updates are limited to CO2 in 1.A, 1.B.2, and 2.A. The removal of FAOSTAT data for sectors 4 (waste) and 5 (other) leads to the use of EDGAR 8.0 data for these sectors and introduces changes in the TP time series as well.
  • For the “country reported priority” time series we have new data covering all sectors for several countries. This leads to changed data in many sectors for these countries. For 2022 and where country reported data are missing the CR time-series have changes in the same sectors as the TP time-series.
  • The new Energy Institute data corrects estimates for 2021 and 2022 for many countries, so many countries have changes in energy CO2 for 2021 and 2022.
  • We use EU data and other national inventory data for most AnnexI countries this year rather than data submitted to the UNFCCC, because the latter is not available. For countries with overseas territories this might have an impact, because emissions from these territories might be excluded for the EU reporting but included for the UNFCCC reporting (especially France, (Netherlands territories should not be included in UNFCCC emissions reporting either))
  • We have noticed high differences in waste emissions for EU inventory countries.
